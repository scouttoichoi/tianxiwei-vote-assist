流程 1: 分身多开通道 (Instances) 的添加与配置

为了能够安全地同时多开账号投票而不被 Bugs 封 IP，你需要将账号分配到不同的独立“分身通道”中。每个通道建议分配一个独立的代理网络 (Proxy) 以获得最完美的安全防封保障。

1
新建独立分身

点击左边列表区域右上角的白色按钮 Thêm phiên bản (添加分身)。

2
设置分身名字与独立网络 (Proxy)

填入好记的名称（例如: 多开分身 1），在 Proxy 栏中粘贴您购买的代理地址（支持 ip:端口 或 账号:密码@ip:端口 格式）。点击保存。

3
日常分身管理

在列表中点击分身，该行会以自动高亮显示。点击旁边的铅笔图标可以随时修改网络配置，点击垃圾桶图标可以彻底删除该分身。

⚠️
独立网络代理警告 多开超过 3 个分身时，请务必在分身中配置独享网络代理，严禁所有窗口共用家里的原宽带高频注册，否则会被 Bugs 临时锁死您整条街的网络。

第一步（最关键的先决条件）：通过“下载模板”一键导出别名数据表

IP 熔断机制全自动熔断防封保护

软件内置了高度灵敏的系统保护策略，随时保护您的私人 network 和宝贵的投票大军。

Bugs 对同 IP 高频动作打击极其频繁。一旦您的网络在短时间内发起过多的注册，Bugs 就会锁定并封杀您的网络。 为了防止账号被殃及池鱼造成废号，软件在运行时会持续监督日志输出。一旦任何一个多开窗口捕捉到了 Bugs 返回的 Your IP is temporarily unavailable for membership (您的网络已被临时禁用) 拦截通知，软件将瞬间启动全线自动紧急熔断：强行关闭所有活跃的分身进程和受控浏览器，中断操作，并向您的电脑发送通知，提示您立刻拔插光猫更换网络 IP 以保护资产。

🛠️

日常突发简易排障 (小白备用指南)

遇到 Bugs 封 IP 提示 / "Your IP is temporarily unavailable..."

原因: 代表您的家庭宽带网络因频率超标已被官方临时拉黑。
解决方法: 拔掉您家 Wi-Fi 路由器的电源插头，静置 30 秒再插上，这样您的宽带运营商就会给您自动发一个新的 IP 地址恢复访问。或者在运行分身前打开第三方 VPN 进行网络更换。

刷爱心广告时，提示无法识别、搜索不到任何虚拟机设备

原因: 模拟器的内核调试端口未启用，或被安全管家拦截。
解决方法: 完全关闭模拟器并重新启动一次。进入模拟器设置 -> 开发者选项 -> 找到 "USB 调试" 开关，将它关闭一次再重新打开即可恢复正常连接。
